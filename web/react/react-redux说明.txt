export { Provider, createProvider, connectAdvanced, connect }

Provider属性为一个类,Provider类只允许有一个子元素,来展示我们自己创建的类,props必须赋值store属性
如<Provider store={store}>
    <App />
  </Provider>
Provider类提供给App子类content属性,值为{ [storeKey]: this[storeKey], [subscriptionKey]: null },storeKey的值默认为'store',subscriptionKey的值默认为{storeKey+'Subscription'}.


createProvider,运行 createProvider 函数可生成并返回Provider类,createProvider(storeKey,subscriptionKey),可修改storeKey和subscriptionKey


connectAdvanced,运行 connectAdvanced 函数返回 wrapWithConnect 函数,运行 wrapWithConnect 函数返回Connect类.
